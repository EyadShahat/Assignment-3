# Technical Documentation

## Project Overview
This enhanced portfolio website is designed to demonstrate interactivity, data handling, and AI-assisted development.  
It is a continuation of Assignment 1 with new JavaScript-driven features and animations.

## Features
- **Personalization** → Users can set and save their name using localStorage.  
- **Dynamic Data** → Fetches random facts from a public API with loading and retry states.  
- **Dark/Light Theme** → Theme preference stored and loaded from localStorage.  
- **Projects Section** → Live search, filter buttons, and collapsible details for each project.  
- **Form Validation** → Inline error messages and success feedback.  
- **Animations** → Smooth reveal transitions using IntersectionObserver.  

## Technologies Used
- **HTML5** → Structure and content  
- **CSS3** → Styling, layout, animations, transitions  
- **JavaScript (ES6)** → Logic for interactivity, data handling, and feedback  
- **API Integration** → Public “Quotable” API for random quotes  
- **AI Assistance** → ChatGPT used for debugging, structure suggestions, and optimization  
- **Git & GitHub** → Version control and repository hosting
