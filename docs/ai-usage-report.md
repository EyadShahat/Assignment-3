
# AI Usage Report

## Tools Used
- **ChatGPT (OpenAI)**

## Use Cases
- Assisted in debugging JavaScript features such as the dark mode toggle, API fetch, and localStorage.  
- Helped design and refine the logic for live search, filters, and collapsible project sections.  
- Suggested form validation techniques and feedback handling.  
- Provided structure and examples for documentation writing.  

## Benefits
- Faster troubleshooting and problem-solving.  
- Improved understanding of responsive layouts and data handling.  
- Clearer logic flow in JavaScript functions.  

## Challenges
- Some AI-suggested solutions needed modification to fit project requirements.  
- API calls required manual handling for error states and retries.  
- Needed to adapt code for performance and consistency across browsers.  

## Learning Outcomes
- Enhanced problem-solving skills with AI guidance.  
- Learned how to integrate public APIs and manage errors responsibly.  
- Improved ability to evaluate and modify AI suggestions.  

## Responsible Use
- AI was used only to **assist** in fixing errors, suggesting methods, and improving interactivity.  
- All final code was written, tested, and customized manually.  
- Documentation was reviewed and edited to ensure clarity and originality.
